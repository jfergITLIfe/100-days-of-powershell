#  100 Days of PowerShell

Welcome to my 100 Days of PowerShell challenge.

For the next 100 days, I’m committing to learning, practicing, and documenting PowerShell — one day at a time.

The goal isn’t perfection.  
The goal is consistency.

---

## 🎯 Objectives

- Build real PowerShell automation skills
- Strengthen my IT / Security scripting foundation
- Create a public portfolio of progress
- Document lessons, mistakes, and improvements
