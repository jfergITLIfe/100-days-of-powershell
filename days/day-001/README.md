# Day 001 – Exploring Get-Command & Get-Help

## 🎯 Goal
Understand how to discover and research commands in PowerShell.

---

## 🔎 Commands Used


```powershell
Get-Command
Get-Help Get-Process
Get-Help Get-Process -Examples
Get-Help Get-Process -Full
```

🧠 What I Learned

* PowerShell has thousands of built-in commands.
* Get-Help is extremely detailed.
* Using -Examples is the fastest way to learn.
* Everything in PowerShell outputs objects, not just text.

⚠ Observations
* Update-Help
* The Update-Help cmdlet downloads the newest help files for the Powershell modules.
