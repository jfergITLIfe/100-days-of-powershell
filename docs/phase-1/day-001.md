# Day 001 — What is PowerShell?

## Overview

PowerShell is a command-line shell and scripting language built by Microsoft. Think of it as a supercharged Command Prompt — but instead of running basic commands, you can automate tasks, manage systems, and interact directly with Windows, Azure, and beyond.

Here's what makes it different from the old `cmd.exe`: PowerShell doesn't just return text. It returns **objects**. That means when you ask for a list of running processes, you don't get a wall of text — you get structured data you can filter, sort, and pipe into other commands. That distinction matters more and more as you progress.

IT pros, sysadmins, and security analysts use PowerShell every single day. It runs on Windows, macOS, and Linux. If you're heading into any technical IT role, you will encounter this tool. Better to meet it on your terms.

---

## Your first command

Open PowerShell and run this:

```powershell
Write-Host "Hello, World!"
```

You should see `Hello, World!` printed to the terminal. That's it — you just ran your first **cmdlet** (pronounced "command-let"). Cmdlets are the building blocks of PowerShell. They follow a `Verb-Noun` naming pattern: `Write-Host`, `Get-Process`, `Stop-Service`. Once that pattern clicks, the language starts to feel readable even before you know it.

---

## A few things worth knowing on Day 1

PowerShell has two major versions in the wild:

- **Windows PowerShell (5.1)** — ships with Windows, lives at `C:\Windows\System32\WindowsPowerShell\v1.0\powershell.exe`
- **PowerShell 7+** — the modern, cross-platform version. This is what we use in this challenge.

If you haven't installed PowerShell 7 yet, grab it [here](https://github.com/PowerShell/PowerShell/releases). It installs alongside 5.1 — nothing gets overwritten.

---

## What you learned

- PowerShell is a shell and scripting language that works with objects, not just text
- Cmdlets follow a `Verb-Noun` pattern
- There are two major versions — we use PowerShell 7+
- `Write-Host` prints output to the terminal
