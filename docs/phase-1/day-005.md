# Day 005 — Arithmetic & String Operators

## Overview

Operators are how you do work on data. Math, comparisons, combining strings — it all goes through operators. Nothing exotic here, but knowing the comparison operators cold is something you'll thank yourself for in every conditional and filter you write from Day 21 onward.

---

## Arithmetic operators

```powershell
$a = 20
$b = 6

$a + $b    # 26
$a - $b    # 14
$a * $b    # 120
$a / $b    # 3.333...
$a % $b    # 2  (modulo — remainder after division)
```

---

## Comparison operators

PowerShell comparison operators use letter abbreviations, not symbols:

```powershell
5 -eq 5      # True  — equal
5 -ne 4      # True  — not equal
10 -gt 5     # True  — greater than
5 -lt 10     # True  — less than
10 -ge 10    # True  — greater than or equal
5 -le 10     # True  — less than or equal
```

---

## String operators

```powershell
# Concatenation
$first = "Power"
$second = "Shell"
$first + $second           # PowerShell

# Repetition
"-" * 40                   # ----------------------------------------

# Contains (case-insensitive by default)
"PowerShell" -like "*Shell*"    # True
"PowerShell" -match "^Power"    # True (regex)
```

---

## Logical operators

```powershell
$true -and $false    # False
$true -or $false     # True
-not $true           # False
```

---

## What you learned

- Arithmetic operators: `+`, `-`, `*`, `/`, `%`
- Comparison operators: `-eq`, `-ne`, `-gt`, `-lt`, `-ge`, `-le`
- String operators: `+` for concat, `*` for repeat, `-like` for wildcards, `-match` for regex
- Logical operators: `-and`, `-or`, `-not`
