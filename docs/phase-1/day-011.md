# Day 011 — Working with Strings

## Overview

Almost everything you touch in IT work comes back as a string at some point — log lines, usernames, file paths, IP addresses, API responses. Knowing how to manipulate strings fluently is non-negotiable. This is a workhorse day.

---

## The essential string methods

```powershell
$text = "  Hello, PowerShell World!  "

# Length
$text.Length

# Uppercase / Lowercase
$text.ToUpper()
$text.ToLower()

# Trim whitespace
$text.Trim()
$text.TrimStart()
$text.TrimEnd()

# Does it contain something?
$text.Contains("PowerShell")     # True

# Does it start or end with something?
$text.StartsWith("Hello")        # False (there's leading whitespace)
$text.Trim().StartsWith("Hello") # True

# Replace
$text.Replace("World", "Universe")

# Split into an array
"one,two,three".Split(",")

# Join an array back into a string
"one","two","three" -join " | "

# Substring — extract part of a string
"PowerShell".Substring(0, 5)     # Power

# Index of a character
"PowerShell".IndexOf("S")        # 5
```

---

## String formatting

```powershell
# -f operator (like printf)
"My name is {0} and I am {1} years old." -f "Jacob", 30

# Padding for alignment
"{0,-20} {1,10}" -f "Service Name", "Status"
```

---

## What you learned

- Core string methods: `Trim`, `ToUpper/Lower`, `Replace`, `Contains`, `Split`, `Substring`
- `-join` combines arrays into strings
- `-f` formats strings with positional placeholders
